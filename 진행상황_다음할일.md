## 현재 프로젝트 상태

현재 프로젝트는 `Flutter + Flame` 기반의 2D 점프게임 프로토타입 단계다.  
메인 방향은 `슈퍼마리오형 횡스크롤 플랫폼 게임`이고, 이제는 기반 공사보다  
`실제 게임 콘텐츠를 붙여가는 단계`에 더 가까워졌다.

기준 화면은 아래처럼 유지한다.

- 비율: `16:9`
- 해상도: `1280 x 720`
- 가로형 고정 뷰포트
